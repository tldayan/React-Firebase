# Realtime Todo Board

## 🚀 Features
- Add, update, delete tasks
- Realtime sync using Firebase Realtime Database
- Task filtering (Todo, In Progress, Done)
- Realtime task counters
- Inline editing

## 🛠 Tech Stack
- React (TypeScript)
- Firebase Realtime Database
- Vite
